Core Activity Support
데이터 무결성 확보

Foreign Key 제약 조건을 통해 고객 정보와 활동 로그 간의 일관성을 유지하며, Cascade 정책으로 효율적인 데이터 관리를 실현했습니다.

확장 가능한 스키마

JSONB 타입을 활용해 설문(Votes)이나 기기 정보(Bugs) 같은 비정형 데이터를 유연하게 수용할 수 있도록 설계했습니다.

보안 및 성능 최적화

Supabase RLS(Row Level Security)를 고려한 UUID 기반 PK 설계로 사용자별 데이터 접근 권한을 물리적으로 격리했습니다.
